Fixtures

Selecting the fixtures tab will bring up a list of fixtures for todays date. These will be seperated into drop down panels corresponding to their respective league.

For each fixture will be a button labelled 'Get Prediction'. Clicking this button will take you to a new page where you can get more match information,
as well as team stats and predictions for the game.

Leagues

The leagues tab will display live league table data, English Premier League by default. You can change which league you want to see by using a drop down box in the top left.

There is another drop down box if you want to switch between overall, home and away tables. Tables can also be sorted by any of the stats by clicking the heading at the top of the column.

Login

By selecting the login option in the top right of the site, the user can login using their Google credentials



API Limitations

As of May 15th 2019, the API package will be downgraded, causing reduced functionality. This means their will no longer be coverage of the English League 1 and the Scottish Premier League.

Additionally, the number of calls to the API will be reduced from 30 to 10. Exceeding this limit will result in data errors and the user will have to wait till the requests are refreshed.

API calls are made as follows:

  • 1 call when opening the fixtures page and another when changing date
  • 1 call when opening the leagues page and another when changing league
  • 4 calls when generating a match prediction or refreshing the page
